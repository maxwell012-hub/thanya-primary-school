# Thanya Primary School

> Built with [MaxG Site Manager](https://maxgen.site) — Website Operating System for Astro

## Template
`school`

## Development

```bash
# Edit your content
maxg edit company
maxg edit services
maxg edit gallery
maxg edit contact
maxg edit seo

# Preview
maxg preview

# Deploy
maxg deploy
```

## Content Files

All website content lives in the `content/` directory:

```
content/
├── company.json       # Business info, contact details, social links
├── services.json      # Services offered
├── gallery.json       # Gallery images
├── testimonials.json  # Client testimonials
├── contact.json       # Contact form settings
├── seo.json           # SEO metadata
└── pages/
    ├── home.json      # Home page content
    └── about.json     # About page content
```

> ⚡ Never edit files in `src/` directly — use `maxg edit` commands instead.
